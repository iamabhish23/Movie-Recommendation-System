# Movie-Recommendation-System

This is a simple web application built with Python and Streamlit that provides movie recommendations based on user selection. The recommendation algorithm utilizes precomputed similarity scores between movies.

## Features

- Select a movie from the dropdown menu.
- Click the "Show Recommendation" button to see similar movie recommendations.
- View movie titles and posters for the recommended movies.

## Technologies Used

- Python
- Streamlit
- Pickle
- Requests
- The Movie Database API

## Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/iamabhish23/Movie-Recommendation-System
   ```

2. Install the required dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Run the Streamlit app:

   ```bash
   streamlit run movie_recommender.py
   ```

## Usage

- Upon running the Streamlit app, open the provided URL in your web browser.
- Select a movie from the dropdown menu.
- Click the "Show Recommendation" button to generate movie recommendations.
- View the recommended movies along with their titles and posters.

## Credits
